# Chronic Disease Risk Prediction Framework

This repository contains a modular machine learning framework for predicting chronic diseases using structured clinical data.

The goal of this project is to develop scalable, reproducible, and deployable ML systems for early disease risk detection.

---

## Implemented Modules

### 1️⃣ Diabetes Risk Prediction
- Dataset: Pima Indians Diabetes Dataset
- Model: Balanced Logistic Regression
- Accuracy: 72%
- Recall (Diabetic): 72%
- ROC-AUC: 0.81
- Deployment: Streamlit Web Application
